Hey guys, starting today, I will be updating daily until Wednesday (i will try to) because I started a Linux challenge on Wednesday. A few days ago, I decided to switch from elementaryOS to Kubuntu, and finally try out KDE. And the results? Kubuntu is my new favorite desktop environment. Every other DE i've used has always ended up with me feeling like I'm using an alien OS. Not anymore though, with Kubuntu, problems are minimal, and the experience feels like home.

So I started customizing, I tried making it look like macOS, but after using latte dock, I decided not to. Then I tried a Windows 11 style- that didn't go well either. So I just said screw it, I'm making my own thing. I chose a One Dark-styled theme (the theme I use for VSCode) and the result is now this:



I think it looks fine now. Some problems I ran into along the way was the KDE store not working from the settings tab, to quickly install themes and such. But I just installed them from the website as files, so it's fine.

Later on, I got my games working. First was Minecraft, with all the mods, resource packs, and Fabric, all working. Then Geometry Dash, which was the first time I got MegaHack working on Linux. Minecraft actually plays really smoothly (since it's natively supported) but with Geometry Dash supported only with Proton, sometimes I experience slowdowns and lag. Kinda annoying but it's playable. The FPS counter is actually higher than on Windows!

After all this happened, I finally decided to start the Linux Challenge. So we start with Day 1 now.

Day 1



I played around with the KDE system sounds, and despite them being from 2008, they still sound extremely modern and as good as any other operating system sounds I've heard.

One issue I later came across was the keyboard. On a Windows keyboard, the Function (FN) keys do something completely different, so I had to switch to Apple Keyboard mode, remap some keys, and swap the alt and meta (Windows/Command) key.

Then I had Python classes (easy) and installed Zoom. It was UGLY

Oh, and some other words that I haven't mentioned: Spectacle (the screenshotting tool) is amazing. It might not be Snipping Tool, but it's way better than every other screenshotting tool i've used on Linux. 9/10. And the wobbly windows effect is cool

I got VSCode, and thank god for Settings Sync, I didn't have to deal with resetting up VSCode. All my extensions, themes, and settings were here. Nothing new.

Later, I noticed that my 18GB root partition was almost becoming full, so I had to resize it. Only thing was, most partitions were in use, so I couldnt move/resize much. So after class, I got GParted on my flash drive, resized everything, and now my root partition is 26GB. Also, the GParted desktop environment looked like garbage. (I know it's supposed to be just one program, but still, looks like garbage)

Then I tried Thunderbird for the first time. It's the same as the GParted live USB. LOOKS LIKE GARBAGE. For me, it's web or nothing for email (unless you're talking the Apple Mail app, that one looks very good and I use it on my iPhone)

Day 2



I started the day with getting MegaHack working on Geometry Dash. After multiple attempts, it finally worked. Big good job

Then I donated 2 euros to KDE because I really love the DE

After that, I realised I didn't have a video recording software. On Windows, I would use xbox game bar (because OBS was very slow and recorded at like 5fps [I dont have a good computer]) so my friend recommended Kazam to me. Overall, it's just as good as xbox game bar, but some problems include the performance. On Minecraft, it worked fine, no issues. But on Geometry Dash, it would slow down my game exponentially, and causes massive input delay. The FPS count stays the same for some reason.

Which leads us to now, which I had some problems with Git at first with VSCode, but that has been resolved. And that's it! Stay tuned for tomorrow (Day 4!) oh wait i completely forgot, the challenge lasts for 1 week
